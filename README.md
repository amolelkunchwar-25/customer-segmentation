# 🛍️ Customer Segmentation Project

## 📌 Overview
Segmented 200 customers into 4 groups using K-Means 
clustering based on behavioral and demographic data.

## 🛠️ Tools Used
- Python, Pandas, NumPy
- Scikit-learn (KMeans, PCA, StandardScaler)
- Matplotlib, Seaborn

## 📊 Features Analyzed
- Age
- Annual Income
- Spending Score
- Purchase Frequency
- Average Order Value

## 👥 Segments Found
| Segment | Description |
|---------|-------------|
| Budget Shoppers | Low income, low spending |
| High Value Loyalists | High income, high frequency |
| Occasional Spenders | Medium income, low frequency |
| Young Explorers | Young age, variable spending |

## 📁 Files
| File | Description |
|------|-------------|
| customer_segmentation.ipynb | Main Jupyter notebook |
| customer_segments.csv | Output data with segment labels |

## ✅ Result
- Optimal Clusters : 4
- Method : K-Means + Elbow Method + PCA
